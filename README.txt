CashBoxes Pro (PWA) — نسخة احترافية للموبايل

تشغيل سريع:
1) ارفع كل الملفات (بدون مجلدات) إلى جذر الاستضافة (GitHub Pages أو أي استضافة ثابتة).
2) تأكد من تفعيل Firebase Anonymous Auth (أو Email/Password لو تحب).
3) أنشئ Firestore Database، وطبّق قواعد الأمان (مذكورة في التوثيق).
4) Cloudinary: تأكد من وجود Unsigned Upload Preset باسم Joodkids داخل Cloud name: dthtzvypx
5) افتح الرابط من الموبايل ثم اختر "تثبيت" لتثبيت التطبيق كـ PWA.

ميزات أساسية:
- دارك/فاتح/تلقائي
- Drawer احترافي (يغلق عند الضغط خارجها)
- عند الضغط على الصندوق يظهر كشف حساب مع فلترة بالتاريخ + حساب الرصيد الافتتاحي لنفس التاريخ
- إضافة/تعديل/حذف معاملات (مع حالة Void)
- استيراد/تصدير Excel (XLSX) و CSV
- تقارير: شهري + أفضل الحسابات + رسم بياني Canvas
- دعم Offline عبر Service Worker

ملاحظة أمنية:
- لا يتم تخزين كلمة مرور داخل الملفات. استخدم شاشة الإعدادات لتسجيل الدخول إذا فعلت Email/Password.
